\section{apache/httpd}

\textit{Az apache beállítása (SSL támogatással)}

1.) Az elsõ lépések megkövetelik a root jogkört, ezért használja a {\tt su -} parancsot, hogy megkapja a root-jogokat.


2.) Alapértelmezetten az Apache szerver nem fut. Ez megváltoztatható a következõ

{\tt service httpd add}

paranccsal.

3.) Mivel nem akarjuk újraindítani a rendszert, ezért manuálisan indítjuk:

{\tt root@vmhome:~# service httpd start}

{\tt Starting Apache web server (without SSL)        [  OK  ]}

4.) Ha SSL támogatást is akarunk:

{\tt root@vmhome:~# cd /etc/httpd/conf/}

{\tt root@vmhome:/etc/httpd/conf# sh mkcert.sh}

{\tt Signature Algorithm ((R)SA or (D)SA) [R]:}

Fogadja el az alapértelmezett RSA aláírás algoritmust.

Néhány mezõt ki kell tölteni, néhányat üresen is hagyhat.
Ha '.'-adja meg, akkor a mezõ üres marad.


{\tt 1. Country Name             (2 letter code) [XY]:}

Adja meg a kétbetûs országkódot : (pl. US)

{\tt 2. State or Province Name   (full name)     [Snake Desert]:}

Adja meg az államot.

{\tt 3. Locality Name            (eg, city)      [Snake Town]:}

Adja meg városa, faluja nevét.


{\tt 4. Organization Name        (eg, company)   [Snake Oil, Ltd]:}

Szervezetünk neve jön ide.


{\tt 5. Organizational Unit Name (eg, section)   [Webserver Team]:}

A részleg neve.

{\tt 6. Common Name              (eg, FQDN)      [www.snakeoil.com]:}

\textbf{Fontos:} Itt létezõ címet adjunk meg, különben hibajelentést kap a webböngészõjébe.
                           

{\tt 7. Email Address            (eg, name@FQDN) [www@snakeoil.dom]:}

Általában ide a webmester email címét adjuk meg (\textit{webmester@domain.hu})

{\tt 8. Certificate Validity     (days)          [365]:}

Általában egy év jó választás.

Utána kiválasztjuk az igazolvány verzióját:

{\tt Certificate Version (1 or 3) [3]:}

Az alapértelmezett 3-as rendben lesz, nyomja le az entert.


A következõ lépésben megkezdjük a privát kulcs kódolását.

{\tt Encrypt the private key now? [Y/n]:}

Ezek a kulcsok \textit{nem} lesznek olvashatóak a felhasználók számára, úgyhogy ezt a lépést kihagyhatja.



A következõ fileok lettek létrehozva: {\tt /etc/httpd/conf/ssl.key/server.key} (ez a file maradjon privát),
 {\tt /etc/httpd/conf/ssl.crt/server.crt} és {\tt /etc/httpd/conf/ssl.csr/server.csr}.


5.) Most újra kell indítani az apache-t:

{\tt root@vmhome:~# service httpd restart}

{\tt Stopping Apache web server [  OK  ]}

{\tt Starting Apache web server [  OK  ]}

6.) Most meggyõzõdhetünk a parancsok sikerességérõl:

{\tt vmiklos@vmhome:~\$ elinks https://localhost/}

Ez meg kell hogy mutassa az alapértelmezett weboldalt SSL protokollon keresztül. :)
